Bot Discord com API

Bot avançado com comandos inteligentes, integração via API REST, autenticação e menus interativos.

Discord.js + REST APIs

Visão Geral

Desenvolvimento de bots para Discord utilizando JavaScript e Node.js, com foco em integração de APIs externas, sistemas de autenticação e interfaces interativas. Os bots foram utilizados principalmente para gerenciar a comunidade da Lykos Province e em projetos freelance.

Funcionalidades Implementadas

  • Sistema de Comandos: Comandos slash (/) e prefixados com sistema de permissões por cargo.
  • Integração com APIs: Conexão com APIs REST externas para buscar dados em tempo real.
  • Autenticação: Sistema de verificação de usuários com integração ao banco de dados MySQL.
  • Menus Interativos: Botões, select menus e modais para melhor experiência do usuário.
  • Sistema de Tickets: Abertura e gerenciamento de tickets de suporte.
  • Logs e Moderação: Sistema completo de logs de ações e ferramentas de moderação.

Arquitetura

Runtime

Node.js como ambiente de execução, com Discord.js para interação com a API do Discord.

APIs REST

Integração com múltiplas APIs externas usando Axios para requisições HTTP.

Persistência

MySQL para armazenamento de dados de usuários, configurações e logs.

Deploy

Hospedagem em VPS Linux com PM2 para gerenciamento de processos.

// Comando de verificação com integração MySQL
client.on('interactionCreate', async (interaction) => {
  if (!interaction.isCommand()) return;
  
  if (interaction.commandName === 'verificar') {
    const user = await db.query(
      'SELECT * FROM users WHERE discord_id = ?',
      [interaction.user.id]
    );
    
    if (user) {
      await interaction.member.roles.add(VERIFIED_ROLE);
      await interaction.reply({ 
        content: 'Verificação concluída!',
        ephemeral: true 
      });
    }
  }
});

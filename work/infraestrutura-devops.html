Infraestrutura DevOps & CI/CD

Implementação de infraestrutura escalável com pipelines automatizados, containerização e deploy contínuo.

Docker + VPS + CI/CD

Visão Geral

Projeto contínuo de implementação e manutenção de infraestrutura DevOps para diversos serviços. Inclui configuração de servidores VPS, containerização com Docker, pipelines de CI/CD para deploy automatizado, e integração com serviços de CDN e proteção como Cloudflare.

Componentes da Infraestrutura

  • VPS Linux: Servidores Contabo configurados com Ubuntu Server, otimizados para performance.
  • Docker: Containerização de aplicações para isolamento e portabilidade.
  • Docker Compose: Orquestração de múltiplos containers com configuração declarativa.
  • Git Workflows: Fluxos de trabalho com branches, PRs e code review.
  • CI/CD Pipelines: Automação de build, teste e deploy a cada push.
  • Cloudflare: CDN, SSL, proteção DDoS e DNS management.

Pipeline de Deploy

1. Git Push

Código é enviado para o repositório Git (GitHub), disparando o pipeline.

2. Build & Test

Aplicação é construída e testes automatizados são executados.

3. Docker Build

Imagem Docker é criada com a nova versão da aplicação.

4. Deploy

Container é atualizado no servidor VPS automaticamente.

Exemplo: docker-compose.yml

version: '3.8'
services:
  n8n:
    image: n8nio/n8n
    restart: always
    ports:
      - "5678:5678"
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=${N8N_USER}
      - N8N_BASIC_AUTH_PASSWORD=${N8N_PASS}
    volumes:
      - n8n_data:/home/node/.n8n
      
  mongodb:
    image: mongo:6
    restart: always
    volumes:
      - mongo_data:/data/db
      
volumes:
  n8n_data:
  mongo_data:
